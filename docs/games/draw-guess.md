# 你画我猜

## 概述

轮流作画、聊天猜词的聚会游戏。每位玩家当一次画家，猜对得分，积分最高者获胜（可多局）。

## 人数

- 最少 2 人，最多 8 人（建议 4–8 人）
- 不支持电脑玩家

## 流程

1. **选词**（10 秒）：画家从 3 个候选词中选一个，超时随机。
2. **作画**（90 秒）：画家在画布上绘画；其他人看到字数提示（如 `_ _ _`）和实时笔迹。
3. **猜词**：猜家在聊天框输入答案，完全匹配即猜中。
4. **结算**：时间到或全员猜中后公布答案，展示本轮得分，切换下一位画家。
5. **结束**：每人当过画家一局后结束，可「再来一局」。

## 得分

| 情况 | 得分 |
|------|------|
| 第一个猜中 | 猜者 +10，画家 +5 |
| 后续猜中 | 猜者 +5 |
| 画家 | 不能猜词 |

## 防作弊

- 词语仅画家可见（作画阶段）
- 已猜中玩家的发言，对未猜中者不可见，直至回合结束

## 词语包

- **官方分类**：动物、影视、日常、运动等，可建房时多选
- **个人词库**：在「词语包管理」页维护，建房时可勾选
- **本局专用词**：房主在房间设置中临时追加
- **自动更新**：服务端按 `WORD_PACK_UPDATE_URL` 定期同步官方 manifest 至数据库

## 环境变量（服务端）

| 变量 | 说明 |
|------|------|
| `WORD_PACK_UPDATE_URL` | 官方 manifest 远程地址 |
| `WORD_PACK_SYNC_INTERVAL_MS` | 同步间隔，默认 24h |
| `WORD_PACK_SYNC_ON_START` | 启动时是否同步，默认 true |
