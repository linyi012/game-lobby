# 达芬奇密码

## 基本规则

1. 每位玩家获得 5 张数字牌（黑白两色，0–11）
2. 轮流出牌，猜测某位对手的手牌序列
3. 将一张牌放入桌面序列的指定位置
4. 系统反馈：精确匹配数 + 颜色/数字部分匹配数
5. 率先清空手牌的玩家获胜

## 本实现要点

- 每位玩家初始 5 张随机牌
- 猜测对象为其他玩家的手牌顺序
- 反馈采用精确 + 部分匹配规则
- 对手手牌对他人隐藏（显示为 `?`）

## 电脑 AI 行为

| 难度 | 行为 |
|------|------|
| 简单 | 常出最后一张牌，位置随意 |
| 普通 | 随机选牌与位置 |
| 困难 | 较合理的牌与位置选择 |
| 专家 | 很少故意失误 |

## 人数要求

- 最少 2 人，最多 4 人
